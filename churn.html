Methodology

How it was built.

📥
Data Collection
Real telecom dataset including customer contract info, network logs, service packages, and relational DB records across thousands of customers.
🔍
Exploratory Data Analysis
Deep EDA on categorical variables (Partner, Dependents, Services) and continuous variables (Tenure, MonthlyCharges). Correlation heatmaps to identify key drivers.
⚙️
Feature Engineering
Data cleaning, feature selection, stratified train/test splitting to maintain class balance. Feature scaling and categorical encoding with scikit-learn pipelines.
📊
Model Evaluation
10-fold cross-validation for robust accuracy verification. Evaluated using confusion matrices, ROC-AUC curves, precision, and recall scores for each model.
# Model Training Pipeline from sklearn.ensemble import RandomForestClassifier from sklearn.model_selection import cross_val_score, StratifiedKFold from sklearn.metrics import roc_auc_score, confusion_matrix # Best performing model rf_model = RandomForestClassifier( n_estimators=100, max_depth=10, random_state=42 ) # 10-fold cross-validation cv = StratifiedKFold(n_splits=10, shuffle=True) scores = cross_val_score(rf_model, X_train, y_train, cv=cv) rf_model.fit(X_train, y_train) auc = roc_auc_score(y_test, rf_model.predict_proba(X_test)[:,1]) print(f"Accuracy: {scores.mean():.2f}") # → 0.85 print(f"ROC-AUC: {auc:.2f}") # → 0.88
