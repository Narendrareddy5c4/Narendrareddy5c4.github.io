Features

Everything a modern web app needs.

🔐
Google OAuth2 Authentication
Secure user registration and login via Google OAuth2. Each user can only add, edit, or delete their own items — no one else's.
🗂️
Full CRUD Operations
Create, Read, Update, and Delete chair categories and individual chair products. Full ownership model — users manage their own data.
🔌
REST JSON API
Three public JSON endpoints expose catalog data for third-party integrations — all categories, all items, or items by category.
🐍
Python + Flask Backend
Clean Python/Flask backend with SQLAlchemy ORM for database management. SQLite for lightweight, portable data storage.
📱
Responsive Design
HTML5 and CSS templates that adapt to any screen size — desktop, tablet, or mobile — for a seamless browsing experience.
🚀
Vagrant Deployment
Fully reproducible development environment using Vagrant + VirtualBox. One command to get the entire app running locally.

REST API

Public JSON endpoints.

ChairStore exposes three clean JSON endpoints for third-party integrations. No authentication required — just hit the URL and get structured data back.

GET /chairs/category/json
Returns all chair categories as a JSON array
GET /chairs/all.json
Returns all chairs across every category
GET /chairs/category/{id}/items.json
Returns all chairs within a specific category
# Sample API Response
{ "categories": [ { "id": 1, "name": "Office Chairs", "items": 12 }, { "id": 2, "name": "Gaming Chairs", "items": 6 } ] }
